Browser Technologies

Het vak Browser Tech heb ik uiteindelijk niet gehaald. Ik heb het meerdere keren geprobeerd, maar ik begreep er weinig van. Normaal gesproken zijn HTML en CSS talen die ik het makkelijkst schrijf, javascript is mijn valkuil, maar tijdens dit project waren ook deze lastig. Dit maakte het voor mij erg onoverzichtelijk en ik wist niet goed waar ik moest beginnen. Het had me denk ik wel geholpen als we iets meer lessen hadden gehad in plaats van het alleen zelf uitzoeken. Dit was voor de andere waarschijnlijk prima, maar voor mij werkte dit niet goed.
