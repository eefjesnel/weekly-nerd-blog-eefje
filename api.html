Application Programming Interfaces

API vond ik net als HCD lastig. Wederom vond ik het moeilijk om zo snel over te stappen naar een nieuwe vorm van programmeren. Ik had hiervoor nog nooit een framework gezien, laat staan gebruikt. Ik opende het voorbeeldfolder en ik was meteen de draad kwijt. Ik merkte dat het niet uitmaakte wat ik deed, het duurde een eeuwigheid voordat ik het werkend kreeg. Uiteindelijk ben ik niet meer verder gegaan omdat ik meer stress kreeg dan dat er iets in de browser gebeurde. Verder was het een goed en interessant vak, alleen wat te hoog gegrepen voor mij.
