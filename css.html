CSS To The Rescue

CSS to the rescue was mijn favoriete vak tot nu toe in deze minor. We hadden veel vrijheid en ik heb echt heel erg veel geleerd van alle “cursussen” waar je je voor kon inschrijven. Ik vond het nog vrij lastig om de vele nieuwe css toe te passen en niet te verdrinken in de hoeveelheid. Ik heb wel gemerkt dat ik wat duidelijker moet gaan werken, meer comments, structureel en kortere code. Toen ik bij de herkansing kwam, kon ik mijn code met moeite terugvinden en was het niet altijd even duidelijk wat wat was.
